== Description ==

**Create customer contact lists and more with this beautiful export utility!**

Export to Excel, CSV, or even print directly from the screen.

Save your configuration for 1-click exports.

Incredibly easy way to export WooCommerce data!

1. Choose the type of data you want to export.
1. Select the fields.
1. Filter the data.
1. Let's Export!
1. Save the config and run it later with a single click.

Compatible with WooCommerce 2.6 and higher, including 3.0+.

> **The <a href="https://pimwick.com/pw-woocommerce-lets-export">Pro version</a> includes more fields and supports ALL custom fields from any 3rd party plugins!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pw-woocommerce-lets-export` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Pimwick Plugins -> PW Let's Export!
4. Save the config and run it later with a single click.
